SUPPORT & TROUBLESHOOTING
================================================================================

COMMON ISSUES:
--------------

Issue: Dashboard doesn't load
Fix: Check that all files are present in correct folders

Issue: Charts don't show
Fix: Check internet connection (for CDN libraries)

Issue: Data doesn't update
Fix: Hard refresh browser (Ctrl+F5 or Cmd+Shift+R)

Issue: Export doesn't work
Fix: Check browser popup blocker settings

GETTING HELP:
-------------
1. Check README.md for answers
2. Check browser console (F12) for error messages
3. Review DATA-UPDATE-INSTRUCTIONS.md
4. Contact Athenian Group support
